# salery-prediction-
Salary Prediction Machine Learning Project

#Overview

This project predicts employee salaries using Machine Learning based on real-world data from Kaggle. It uses regression techniques to estimate salary based on features like experience level, employment type, job role, and company size.

Dataset

Dataset used: Data Science Job Salaries

Technologies Used
Python
Pandas
NumPy
Scikit-learn
🚀 Project Workflow
Data Collection
Data Preprocessing
Handling categorical variables using Label Encoding
Feature scaling using StandardScaler
Train-Test Split
Model Training (Linear Regression)
Model Evaluation (MAE)
Salary Prediction
▶️ How to Run
1. Install Dependencies
pip install -r requirements.txt
2. Run Notebook

Open:

salary_prediction_notebook.ipynb

Run all cells step by step.

Linear Regression
📌 Features Used
Experience Level
Employment Type

The model predicts salary (salary_in_usd) based on given inputs.
